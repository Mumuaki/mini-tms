## СТРОГО ОБЯЗАТЕЛЬНО
1. при внесении измнений:
- в код следовать строго поставленной задаче об изменениях, не ломая и не внося никаких других изменений в другие места рабочего кода,
- любые изменения, не связанные с поставленной задачей, вносить только с разрешения пользователя. для таких изменений для получения разрешения пользователя необходимо предоставить разъяснение для чего планируются такие изменения
