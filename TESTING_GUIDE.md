# Руководство по тестированию TMSmini (Gemini-v1)

**Версия**: 1.0
**Дата**: 2025-12-16

Инструкция для ручного тестирования реализованного функционала: Скрапер, Флот, UI.

## 1. Подготовка
- [x] Backend запущен (`http://localhost:8000`)
- [x] Frontend запущен (`http://localhost:5173`)
- [ ] Откройте приложение в браузере: **http://localhost:5173**

## 2. Сценарий: Управление Флотом (My Trucks)
1. Перейдите на страницу **My Trucks**.
2. Нажмите **Add New Truck**.
3. Заполните форму тестовыми данными:
   - **License Plate**: `TEST-001`
   - **Type**: `Тентованный фургон`
   - **Length**: `13.6`
   - **Width**: `2.45`
   - **Height**: `2.7`
   - **Max Payload**: `24000` (кг)
4. Нажмите **Save Truck**.
5. Убедитесь, что грузовик появился в списке.

## 3. Сценарий: Ручной запуск скрапера (Без грузовика)
1. Перейдите на страницу **Freights** (или Dashboard -> Freights).
2. Нажмите зеленую кнопку **Open Browser**.
   - *Ожидание*: Должно открыться окно Chromium.
   - *Действие*: В открывшемся окне авторизуйтесь на `platform.trans.eu` вручную. Оставьте окно открытым!
3. В TMS заполните фильтры:
   - **Origin**: `PL, 00-001` (Варшава)
   - **Destination**: `DE, 10115` (Берлин)
4. Нажмите синюю кнопку **Start Scraper**.
   - *Ожидание*: Кнопка сменится на "Scraping...". В окне браузера начнется магия (ввод данных, поиск).
5. Через 10-15 секунд таблица должна обновиться результатами.

## 4. Сценарий: Скрапинг для Грузовика (Продвинутый)
1. Убедитесь, что у вас выбран грузовик в фильтре **Select Truck** (TEST-001).
2. Поле **Origin** можно очистить (если у грузовика нет GPS, оно останется пустым, тогда введите `PL, Krakow` вручную).
3. Обратите внимание, что фильтр веса в браузере должен автоматически выставиться в **24 т** (исходя из Max Payload грузовика).
4. Запустите скрапер.
5. Проверьте результаты в таблице.
   - Колонка **Cargo** должна содержать вес.

## 5. Сценарий: UI/UX
1. Проверьте сортировку кликом по заголовкам **Distance** и **Price**.
2. Проверьте пагинацию (если записей > 25).
3. Наведите курсор на строки, проверьте hover-эффект.

**Если возникли ошибки:**

- Если браузер закрылся или завис, перезапустите Backend.
